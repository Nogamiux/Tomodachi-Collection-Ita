# Come Editare i Testi

Per verificare file completati e non completati, consulta la tabella relativa hai file. [Clicca qui per Vedere la Tabella](../README.md#tabella-dei-file).

<h4 align="center"> Requisiti: </h4>

- Un Computer Windows Mac o Linux

- Un Editor Come [Visual Studio Code](https://code.visualstudio.com/) o [Sublime Text](https://www.sublimetext.com/) (Raccomandato), NON UTILIZZARE IL BLOCCO NOTE!

⚠️ Nei file di testo sono presente dei caratteri "In VScode sono segnati con un NUL in rosso, mentre su sublime come <0x0>" NON vanno eliminati! 
<p align="center"> ⛔Eliminarli comprometterà l'intero gioco⛔ </p>

⚠️ Attenzione nello scrivere righe troppo lunghe.
la dimensione dello schermo del ds non è molto generosa, righe di testo troppo lunghe potrebberò uscire fuori dallo schermo.
quindi ogni tanto se vedi che la riga e troppo grande rispetto a quella originale, vai a capo o scrivi parole più corte
